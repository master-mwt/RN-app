## Application

## Info

based on TMDb API

## Manual Installation

* `git clone` https://github.com/master-mwt/react-native-app.git
* `cd react-native-app`
* `yarn install`
* `pod install` (ios)
* `npx react-native run-ios` or/and `npx react-native run-android`
